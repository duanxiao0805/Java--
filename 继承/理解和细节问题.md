## 继承的理解,细节

### 子类继承了所有的属性和方法,但是私有属性不能在子类直接访问,必须通过公共的方法去访问

```java
public class A {
    public  int a=1;
    int b=2;
    private int c=3;
}
```

```java
public class B extends A {

}
```
